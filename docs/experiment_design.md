# 实验设计草案

## 实验目标

实验围绕本文三项贡献展开，验证近机作业无人 GSE 在机位近机场景下的结构化通道风险感知、风险分级和分区避让能力。

1. 验证 SOC-UGSE 结构化作业通道及 LTC-UGSE、LSAC-UGSE 区域规则是否能够支撑风险对象占道判断。
2. 验证 YOLO、RGB-D、激光雷达、轮速里程计和 IMU 融合后，能否识别风险对象并获得距离、横向偏移、占道状态和 TTC 等风险特征。
3. 验证风险分类分级模型和分区避让决策矩阵是否符合近机作业安全规则，即纵向通行通道在固定速度巡线过程中允许远离航空器侧受限绕行，横向近机作业通道在固定速度直行过程中禁止任何转向和绕行，前方出现航空器以外的障碍物时执行紧急制动。

实验路径采用简化控制假设：车辆已进入对应通道且初始姿态良好，不把服务车道驶入、定点转向和横向切入作为验证重点。

## 实验平台

实验平台包括上位机、1:10 A320 机位物理模拟平台和 Yahboom R2 ROS 小车。

上位机配置：

- Intel Core i9-13900K
- 64 GB 系统内存
- NVIDIA RTX A4000 GPU
- 16 GB 专用显存

物理模拟平台包括：

- 1:10 A320 航空器模型
- 机坪地皮和机位标线
- 机位红线、黄色中线 / 航空器引导线、白线斑马线 / 等待区标记
- 地勤人员模型
- 有人保障车辆模型
- 多辆可控 ROS 小车，用于扮演无人 GSE

无人 GSE 载体包括：

- Yahboom R2 ROS 小车
- Jetson Orin NX 16G 边缘计算单元
- Astra Pro Plus RGB-D 相机
- YDLIDAR 4ROS 单线激光雷达
- IMU 与轮速里程计
- 可选 DCXGW12030 单目 RGB 相机，用于实验记录、标线辅助感知或冗余视觉输入，不作为 3D 定位前提

## 评价指标

### 感知定位与占道判断指标

- YOLO 检测 Precision、Recall、mAP、F1
- 人员、有人保障车辆、其他无人 GSE、静态障碍物分类别召回率
- RGB-D 深度反投影距离误差
- 激光雷达近距离障碍物确认准确率
- 目标横向偏移误差
- 通道占用判断准确率
- 单帧感知推理时间

### 风险分类分级指标

- 风险等级分类准确率
- 风险等级混淆矩阵
- 风险低估率，即真实高风险被判定为较低风险的比例
- 风险高估率，即真实低风险被判定为较高风险的比例
- Level III / Level IV 高风险召回率
- 风险分级平均计算时间

### 避让决策与执行指标

- 避让决策准确率
- 规则违反次数
- 紧急制动响应时间
- 最小安全距离
- 远离航空器侧受限绕行方向正确率
- 误绕行次数
- 非必要紧急制动次数
- 任务恢复成功率
- 决策到控制执行延迟

## 实验一：多源风险对象感知与通道占用判断实验

目的：验证系统能否识别风险对象，并将检测结果转换到结构化通道坐标关系中。

实验设置：

1. 在纵向通行通道、横向近机作业通道和近机作业区分别布置人员、有人保障车辆、其他无人 GSE 和静态障碍物。
2. 设置目标位于通道内、通道边界、通道外接近和远离通道等不同位置。
3. 控制车辆低速通过场景，采集 RGB-D 图像、激光雷达、IMU、轮速里程计和 rosbag 数据。

记录内容：

- YOLO 检测框、类别和置信度
- RGB-D 深度反投影后的目标距离与横向偏移
- 激光雷达近距离障碍物确认结果
- 目标所属通道区域和占道状态
- 单帧处理时间

评价方法：

- 与人工标注目标类别对比，计算 Precision、Recall、mAP 和 F1。
- 与人工测量距离或标定距离对比，计算距离误差和横向偏移误差。
- 与人工标注的占道状态对比，计算通道占用判断准确率。

## 实验二：风险分类分级实验

目的：验证风险量化模型能否根据目标类别、通道区域、距离 / TTC 和占道状态输出合理风险等级。

风险等级可先采用四级划分：

| 等级 | 含义 | 典型条件 |
| --- | --- | --- |
| Level I | 正常 | 无目标占道，或目标远离通道 |
| Level II | 注意 | 目标接近通道边界，距离较远或 TTC 较大 |
| Level III | 警告 | 目标占用通道，或距离 / TTC 达到危险阈值 |
| Level IV | 高危 | 人员或车辆近距离占道，存在直接碰撞风险 |

实验设置：

1. 构建风险样本库，覆盖不同目标类别、不同通道区域、不同距离、不同相对速度和不同占道状态。
2. 对每组样本进行人工风险等级标注，形成风险分级真值。
3. 使用系统输出风险等级，与人工标注结果进行对比。

记录内容：

- 风险对象类别
- 所属通道区域
- 目标距离、横向偏移和 TTC
- 人工标注风险等级
- 系统输出风险等级
- 计算时间

评价方法：

- 计算风险等级分类准确率和混淆矩阵。
- 单独统计 Level III / Level IV 的召回率。
- 统计风险低估率和风险高估率，重点关注高风险被低估的情况。

## 实验三：风险分级决策矩阵验证实验

目的：验证同一风险对象在不同通道区域内是否触发不同避让动作。

决策矩阵示例：

| 风险等级 | LTC-UGSE 纵向通行通道 | LSAC-UGSE 横向近机通道 / 近机作业区 |
| --- | --- | --- |
| Level I | 固定速度巡线 | 固定速度直行 |
| Level II | 减速观察或保持巡线 | 非航空器目标未在前方占道时固定速度直行 |
| Level III | 远离航空器侧受限绕行或紧急制动 | 非航空器障碍物前方占道时紧急制动 |
| Level IV | 紧急制动 | 紧急制动 |

实验设置：

1. 将同类目标分别放置在 LTC-UGSE 和 LSAC-UGSE 内。
2. 设置目标轻微接近、部分侵入、完全占道和近距离占道等状态。
3. 记录系统输出的风险等级和避让动作。

评价方法：

- 与人工规则表对比，计算决策准确率。
- 统计 LSAC-UGSE 中是否出现不应有的绕行动作。
- 统计 LTC-UGSE 中受限绕行方向是否始终选择远离航空器侧。
- 统计规则违反次数。

## 实验四：分区避让执行实验

目的：验证车辆能否在真实运动过程中执行风险分级避让动作。

### 4.1 纵向通行通道固定速度巡线与受限避让实验

初始条件：车辆已位于纵向通行通道内，车身方向与通道方向基本一致，车辆按固定速度巡线前进。

场景：

- 人员从纵向通行通道边界接近车辆前方。
- 有人保障车辆占据纵向通行通道中心。
- 静态障碍物位于纵向通行通道内，远离航空器侧存在可绕行空间。
- 静态障碍物占用过大，远离航空器侧绕行空间不足。

预期动作：

- Level I 风险下车辆保持固定速度巡线。
- Level II 风险下车辆减速观察或保持巡线。
- Level III 风险且空间充足时，车辆向远离航空器侧受限绕行。
- Level III 风险但空间不足时，车辆紧急制动。
- Level IV 风险下车辆紧急制动。

记录指标：

- 紧急制动响应时间
- 最小安全距离
- 受限绕行方向正确率
- 最大横向偏移
- 规则违反次数

### 4.2 横向近机作业通道禁绕与紧急制动实验

初始条件：车辆已位于横向近机作业通道内，车身方向与通道中心方向基本一致，车辆按固定速度直行，不进行任何转向控制或绕行动作。

场景：

- 人员站立或穿越横向近机作业通道。
- 有人保障车辆占据横向近机作业通道。
- 其他无人 GSE 与本车在近机通道内形成交互。
- 静态障碍物位于近机作业点附近。
- 航空器模型位于通道侧方，仅作为安全边界和背景对象，不作为普通障碍物触发对象。

预期动作：

- 车辆不得执行横向自由绕行。
- 非航空器目标未占用前方通行区域时，车辆保持固定速度直行。
- 非航空器目标进入前方通行区域时，车辆紧急制动。
- Level IV 风险下车辆紧急制动。
- 目标离开通道后，车辆恢复固定速度直行。

记录指标：

- 横向近机通道误绕行次数
- 紧急制动响应时间
- 最小安全距离
- 恢复通行成功率
- 规则违反次数

## 实验五：纵向 / 横向双通道组合验证实验

目的：验证风险分级避让框架能否在纵向通道和横向近机通道两类典型实验路径中稳定执行。

实验流程：

1. 将车辆放置在纵向通行通道起始段，调整车身方向与通道方向一致。
2. 车辆按固定速度巡线前进。
3. 在纵向通道内设置低风险、中风险和高风险目标，验证保持巡线、减速观察、远离航空器侧受限绕行和紧急制动。
4. 将车辆重置到横向近机作业通道起始段，调整车身方向与横向通道方向一致。
5. 车辆按固定速度直行，不执行转向或绕行。
6. 在横向近机通道前方设置航空器以外的占道目标，验证紧急制动；目标清空后，车辆恢复固定速度直行。
7. 设置航空器侧背景对象，验证其不作为普通障碍物触发对象，但持续参与安全边界约束。

本实验不把路径跟踪误差作为主要评价对象，而是重点评价风险识别、风险分级、决策动作和安全规则执行情况。

## 实验六：对比实验与消融分析

目的：验证结构化通道区域规则、多源目标定位与占道判断和风险分级决策对系统安全性的贡献。

对比方案：

1. Baseline：普通固定阈值避障策略，不区分 LTC-UGSE 与 LSAC-UGSE。
2. 无区域规则：保留目标检测和距离估计，但不使用通道区域约束。
3. 无目标类别权重：人员、车辆、静态障碍物采用相同风险权重。
4. 无距离 / TTC 风险量化：仅根据目标是否占道触发动作。
5. 无 RGB-D 深度反投影：仅使用 2D 检测框和激光雷达近距离触发。
6. 横向通道允许普通绕行：取消 LSAC-UGSE 禁绕约束，观察是否产生不符合近机安全规则的横向偏移。
7. 仅固定速度巡线 / 直行控制：不加入风险分级决策矩阵，只保留基础行驶控制。

对比指标：

- 风险等级分类准确率
- 避让决策准确率
- 风险低估率
- 紧急制动响应时间
- 最小安全距离
- 误绕行次数
- 横向近机通道规则违反次数
- 非必要紧急制动次数

## 数据记录建议

每次实验建议记录以下数据：

- 实验编号、场景类型、目标类别和目标区域
- 车辆当前区域
- 目标距离、横向偏移、相对速度和 TTC
- 人工标注风险等级
- 系统输出风险等级
- 系统输出避让动作
- 实际执行动作
- 紧急制动响应时间
- 最小安全距离
- 是否违反区域避让规则

## 风险与补充说明

实验重点应放在风险分类分级与避让规则验证上。纵向通道以固定速度巡线和分类分级避障为核心，横向通道以固定速度直行和非航空器障碍物紧急制动为核心；服务车道驶入、定点转向和作业后撤离只作为流程背景出现。

如果实验数据不足，可优先完成以下闭环：

1. YOLO 风险对象检测数据集与基础指标。
2. RGB-D / 激光雷达目标相对位置估计与通道占用判断。
3. 风险等级人工标注样本库。
4. 风险分级决策矩阵离线验证。
5. LTC-UGSE 与 LSAC-UGSE 两类典型场景的在线避让实验。
