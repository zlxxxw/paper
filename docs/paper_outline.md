# 论文结构草案

本文结构参考第一篇论文“摘要 / 引言 / 相关工作 / 问题建模与总体框架 / 方法分章 / 行为决策 / 实验 / 结论”的组织方式，但研究对象从固定监控视角下的车辆状态感知，转向无人保障车辆自身在无导航地图 / 无 GNSS 条件下的机位结构化通道行驶、分区避让与异常驶离。

## 摘要

摘要建议采用“背景问题-方法框架-核心模块-实验结果-工程意义”的写法。

建议突出以下内容：

- 机场机位保障车辆在近机区域运行时受空间狭窄、航空器安全边界严格、GNSS 信号不稳定和局部导航地图缺失等因素影响。
- 本文提出一种面向机位安全的无人保障车辆结构化通道行驶与分区避让方法。
- 方法利用机位既有标线与作业区域先验，构建服务车道驶入、纵向主线巡行、横向作业通道靠近和异常安全驶离的无人保障作业通道框架。
- 系统结合 YOLO 目标检测、RGB-D 深度反投影、轮速里程计、IMU 和激光雷达，完成通道约束下的目标感知、状态切换和避让决策。
- 实验在 1:10 A320 机位物理模拟平台与 Yahboom R2 ROS 小车上验证任务完成率、路径跟踪误差、避让响应和异常驶离成功率。

## 第一章 引言

第一章应建立研究必要性和问题边界。

建议内容：

- 机场机位是航空器、保障车辆和人员高密度交互区域，车辆误入、滞留、绕行方向错误和异常停滞都可能造成安全风险。
- 传统全局导航依赖地图、GNSS 或外部调度系统，但机位近机区域存在遮挡、信号不稳定和作业空间强约束。
- 对于保障车辆而言，很多任务具有固定流程和明确区域语义，不一定需要开放环境的全局自主导航。
- 本文关注无导航地图 / 无 GNSS 条件下，车辆如何依托机位既有标线和作业通道框架，从服务车道驶入作业区域，并在异常情况下安全驶离。

本章最后明确本文贡献：

1. 提出无图无 GNSS 条件下的机位无人保障作业通道框架。
2. 提出基于既有机位标线锚定的虚拟通道生成与任务状态切换方法。
3. 提出 YOLO 和深度反投影驱动的分区避让与异常驶离策略。
4. 在缩比物理机位平台上验证方法的可行性。

## 第二章 相关工作

参考第一篇论文的写法，本章按问题链条组织，而不是简单堆叠算法。

### 2.1 机场机坪无人保障车辆与地面运行安全

讨论机坪保障作业自动化、无人保障车辆、近机作业安全边界和地面运行风险。

### 2.2 无 GNSS / 弱地图条件下的移动机器人局部导航

讨论 GNSS 不可靠、地图缺失或地图失效条件下的局部导航、视觉 / 激光 / 里程计辅助行驶方法，并指出机场机位场景具有更强的标线和区域规则先验。

### 2.3 结构化场景中的通道生成与路径跟踪控制

讨论基于车道线、地面标线、虚拟轨迹和先验通道的车辆控制方法，包括 Pure Pursuit、Stanley、有限状态机和局部轨迹跟踪。

### 2.4 机位动态障碍物检测与安全避让

讨论 YOLO、RGB-D 深度反投影、激光雷达安全检测、人员 / 车辆识别，以及规则驱动避让策略。

### 2.5 小结

指出现有研究往往偏向全局导航、固定监控感知或一般移动机器人避障，缺乏面向机位既有标线、近机区域规则和异常驶离需求的一体化结构化通道方法。

## 第三章 问题建模与系统总体框架

本章对应第一篇论文中的“问题建模与系统总体框架”，用于从整体上交代系统输入、输出、模块分层和任务流程。

### 3.1 场景对象与任务定义

定义服务车道、机位入口、等待区、红线纵向主线、横向作业通道、近机作业点、航空器安全侧和异常驶离通道。

### 3.2 无图无 GNSS 条件下的局部通行问题建模

将问题定义为：在不依赖全局导航地图和 GNSS 坐标的条件下，车辆根据机位标线、任务先验和局部感知，完成驶入、靠近、避让和驶离。

### 3.3 系统总体架构

建议分为四层：

- 感知层：YOLO 检测、RGB-D 深度反投影、激光雷达距离约束、标线 / 通道边界感知。
- 状态估计层：轮速里程计、IMU、局部感知辅助的相对运动推算。
- 通道与控制层：虚拟通道生成、任务状态机、Pure Pursuit / Stanley 跟踪控制。
- 安全决策层：分区避让、横向禁绕、远离航空器侧绕行、异常驶离。

### 3.4 任务状态机

状态包括服务车道驶入、等待区启动、纵向巡行、转向切入、横向靠近、作业等待、反向撤离、异常驶离和停车避让。

## 第四章 基于机位既有标线的虚拟通道生成

本章是第一个核心方法章，功能类似第一篇论文中的几何校正章节，即先建立后续控制和决策的空间基准。

### 4.1 机位标线与作业区域语义

说明机位红线、黄色中线 / 航空器引导线、白线等待区标记、服务车道边界和近机作业点的作用。

### 4.2 纵向保障主线生成

利用机位红线或其平行偏置线生成车辆纵向巡行参考线，并定义通道宽度、安全边界和航空器侧约束。

### 4.3 服务车道至机位入口驶入通道生成

根据服务车道边界、机位入口和等待区标记生成驶入段，使车辆从服务车道衔接到机位纵向主线。

### 4.4 横向作业通道生成

根据预设转向距离、转向区域、航空器安全边界和近机作业点生成横向作业通道中心线。

### 4.5 异常安全驶离通道生成

在无图、无信号或近机区域突发占道时，优先生成远离航空器侧的安全驶离通道，使车辆退出机位敏感区域。

## 第五章 通道约束下的车辆行驶控制方法

本章对应第一篇论文中的目标检测 / 状态稳定章节，但重点转为车辆自身控制。

### 5.1 轮速里程计与 IMU 辅助的局部运动推算

描述车辆在无 GNSS 条件下如何利用轮速、IMU 和短时运动模型估计相对位移、航向变化和累计行驶距离。

### 5.2 纵向巡行控制

说明车辆沿红线或虚拟纵向主线低速巡行的方法，给出横向误差、航向误差和速度约束。

### 5.3 转向触发与横向切入控制

说明基于预设距离、区域边界或标线触发转向状态切换，并采用预定义轨迹结合 Pure Pursuit / Stanley 完成横向切入。

### 5.4 横向作业通道低速靠近

强调近机区域低速、中心线跟踪、禁止大幅横向偏移和停车等待策略。

### 5.5 原路撤离与异常驶离控制

区分正常作业完成后的原路撤离和异常触发后的安全驶离，说明不同状态下的路径优先级。

## 第六章 YOLO 驱动的分区避让与安全决策

本章对应第一篇论文中的时空语义行为分析章节，重点是区域语义与规则推理。

### 6.1 动态障碍物识别与深度反投影

YOLO 识别地勤人员、有人保障车辆、其他无人保障车辆和静态障碍物；RGB-D 将检测目标反投影到车辆坐标系或局部通道坐标系。

### 6.2 通道区域语义划分

将车辆所处区域划分为服务车道驶入区、纵向主线区、转向切入区、横向作业通道区、近机作业区和安全驶离区。

### 6.3 分区避让规则

纵向主线区允许车辆向远离航空器一侧受限绕行；横向作业通道和近机作业区禁止绕行，检测到障碍物时停车等待；异常驶离时优先选择远离航空器侧通道。

### 6.4 安全状态机与决策逻辑

将目标类别、目标距离、车辆区域、任务状态和等待时间纳入状态机，输出继续行驶、减速、停车、受限绕行或异常驶离决策。

## 第七章 实验与结果分析

参考第一篇论文的实验写法，实验章节采用“平台与数据-评价指标-分模块实验-综合场景实验-小结”的递进结构。

### 7.1 实验环境与平台配置

描述 i9-13900K、64 GB 内存、RTX A4000 16 GB 显存、1:10 A320 机位物理平台、Yahboom R2 ROS 小车、Jetson Orin NX 16G、Astra Pro Plus、YDLIDAR、IMU、轮速里程计和可选 DCXGW12030 单目 RGB 相机。

### 7.2 评价指标

建议指标包括任务完成率、驶入成功率、异常驶离成功率、平均路径跟踪误差、最大横向偏差、停车响应时间、最小安全距离、误绕行次数、避让决策准确率和任务耗时。

### 7.3 虚拟通道生成与驶入实验

验证车辆在无导航地图 / 无 GNSS 条件下从服务车道进入机位纵向主线的能力。

### 7.4 结构化任务链行驶实验

验证服务车道或等待区启动、纵向巡行、横向切入、低速靠近、作业等待和撤离的完整任务链。

### 7.5 分区避让实验

分别测试纵向主线受限绕行、横向通道停车等待、人员进入近机区域、有人保障车辆占道和其他无人车干扰。

### 7.6 异常无图无信号驶离实验

模拟导航地图不可用、GNSS 缺失、局部占道或作业中断，验证车辆能否沿安全通道驶离机位。

### 7.7 对比实验与消融分析

对比无分区避让、横向允许绕行、无虚拟通道约束、无深度反投影或无异常驶离策略等方案。

### 7.8 本章小结

按“通道生成-控制跟踪-避让决策-异常驶离”的逻辑总结实验结果。

## 第八章 结论与展望

总结本文贡献：

- 提出无导航地图 / 无 GNSS 条件下的机位无人保障作业通道框架。
- 提出基于既有机位标线的虚拟通道生成方法。
- 提出通道约束下的结构化行驶控制和状态切换方法。
- 提出 YOLO 与 RGB-D 深度反投影驱动的分区避让和异常驶离策略。

展望：

- 扩展到更多机型和机位布局。
- 提升光照变化、标线磨损和遮挡条件下的通道识别鲁棒性。
- 引入多车协同和调度接口。
- 在真实机场半实物或封闭测试区域开展进一步验证。
