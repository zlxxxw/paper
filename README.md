# 机场机位无人保障车辆结构化通道风险分级与分区避让方法

本项目用于整理和推进论文《机场机位无人保障车辆结构化通道风险分级与分区避让方法》的研究、实验与写作工作。

## 研究定位

本文不研究机场全场景无人车自主导航，也不以开放道路全局路径规划、三维定位或定点转向控制作为核心创新。研究对象限定为机场机位及其邻近服务车道内的近机保障任务：在无导航地图或无 GNSS 信号条件下，无人 GSE 依托机位既有标线和结构化作业通道框架，识别人员、有人保障车辆、其他无人 GSE 和静态障碍物等风险对象，并根据车辆所处区域、目标类别、目标相对位置和占道关系执行分级避让。

## 核心问题

论文主线从“车辆如何按固定路径行驶”转向“车辆在结构化通道内遇到不同风险时如何做出可解释、可约束的避让决策”。核心问题包括：

1. 如何利用机位红线、航空器引导线、服务车道边界、等待区标记和近机作业区域，建立无人 GSE 结构化作业通道及区域语义。
2. 如何融合 YOLO、RGB-D、激光雷达、轮速里程计和 IMU，获得风险对象类别、距离、相对方位、通道占用和短时运动关系。
3. 如何根据风险对象类型、车辆所在区域、距离 / TTC 和占道状态，对风险进行分类分级。
4. 如何将风险等级映射为正常通行、减速观察、远离航空器侧受限绕行、停车等待和紧急制动等避让动作。

## 核心创新点

1. 提出面向近机作业无人 GSE 的结构化作业通道与区域语义模型，将机位运行空间划分为纵向通行通道、横向近机作业通道、近机等待区域和航空器侧安全边界，为风险分级和避让决策提供规则约束。
2. 提出面向 GSE 机位安全的多源语义风险感知方法，融合 YOLO 目标检测、RGB-D 深度反投影、激光雷达距离约束和局部运动推算，识别风险对象并建立其与结构化通道之间的空间占用关系。
3. 提出结构化通道约束下的风险分类分级与分区避让框架，构建“风险对象-通道区域-距离 / TTC-占道状态”耦合决策矩阵，使无人 GSE 在纵向通道内可受限绕行，在横向近机通道和近机作业区内以停车等待为主；Pure Pursuit / Stanley 等控制器仅作为底层动作执行工具。
