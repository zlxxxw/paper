# 机场机位无人保障车辆结构化通道风险分级与分区避让方法

本项目用于整理和推进论文《机场机位无人保障车辆结构化通道风险分级与分区避让方法》的研究、实验与写作工作。

## 研究定位

本文不研究机场全场景无人车自主导航，也不以开放道路全局路径规划、三维定位或定点转向控制作为核心创新。研究对象限定为机场机位及其邻近服务车道内的近机保障任务：假设无人 GSE 已进入对应结构化通道且初始姿态满足直行条件，车辆在纵向通道内按固定速度巡线行驶，在横向近机作业通道内按固定速度直行，并根据车辆所处区域、目标类别、目标相对位置和占道关系执行风险分级与避让决策。

## 核心问题

论文主线从“车辆如何按固定路径行驶”转向“车辆在结构化通道内遇到不同风险时如何做出可解释、可约束的避让决策”。核心问题包括：

1. 如何利用机位红线、航空器引导线、服务车道边界、等待区标记和近机作业区域，建立无人 GSE 结构化作业通道及区域规则。
2. 如何融合 YOLO、RGB-D、激光雷达、轮速里程计和 IMU，获得风险对象类别、距离、相对方位、通道占用和短时运动关系。
3. 如何根据风险对象类型、车辆所在区域、距离 / TTC 和占道状态，对风险进行分类分级。
4. 如何在纵向通道内执行分类分级避障，并在横向通道内对航空器以外的前方障碍物触发紧急制动。

## 核心创新点

1. 提出面向近机作业无人 GSE 的结构化作业通道与区域规则模型，将机位运行空间划分为纵向通行通道、横向近机作业通道、近机等待区域和航空器侧安全边界，为风险分级和避让决策提供规则约束。
2. 提出面向 GSE 机位安全的多源风险对象感知方法，融合 YOLO 目标检测、RGB-D 深度反投影、激光雷达距离约束和局部运动推算，识别风险对象并建立其与结构化通道之间的空间占用关系。
3. 提出结构化通道约束下的风险分类分级与分区避让框架，构建“风险对象-通道区域-距离 / TTC-占道状态”耦合决策矩阵；纵向通道采用固定速度巡线与远离航空器侧受限绕行，横向通道采用固定速度直行且不进行转向或绕行，当前方出现航空器以外的障碍物时触发紧急制动。
