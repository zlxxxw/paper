# 机场机位无人保障车辆结构化通道行驶与分区避让方法

本项目用于整理和推进论文《机场机位无人保障车辆结构化通道行驶与分区避让方法》的研究、实验与写作工作。

## 研究定位

本文不研究机场全场景无人车自主导航，也不以开放道路全局路径规划为核心问题。研究对象限定为机场机位及其邻近服务车道内的局部保障任务：在无导航地图或无 GNSS 信号条件下，车辆依托机位既有标线和无人保障作业通道框架，从服务车道驶入对应作业区域；或在保障过程中突发无图、无信号等特殊情况时，沿结构化安全通道驶离机位，降低对航空器和保障人员的风险。

## 核心任务链

车辆运行被组织为固定化任务链：

1. 服务车道或等待区启动
2. 基于机位标线进入纵向保障通道
3. 沿机位红线或虚拟纵向主线巡行
4. 到达预设转向区域后切入横向作业通道
5. 横向作业通道低速靠近近机作业点
6. 作业完成或异常触发后沿安全通道撤离机位

## 核心创新点

1. 提出等待区起点下的机位结构化通道行驶框架，将机位局部运行组织为固定化、可工程实现的任务链。
2. 提出无导航地图 / 无 GNSS 条件下基于真实机位标线锚定的虚拟通道生成方法，利用机位红线、等待区白线、航空器引导黄线、服务车道边界和近机作业点，在 ROS 控制系统中生成虚拟纵向主线和横向作业通道。
3. 提出 YOLO 目标识别驱动的分区避让方法，依据车辆所处区域和目标类型执行差异化避让：纵向主线允许受限绕行，横向通道禁止绕行并停车等待。
